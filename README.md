# Poolside

A pool-management web app for water chemistry, equipment status, maintenance, and service history.

## Project layout

- `app/` — website
- `notes/` — Obsidian vault and product documentation
- `.openai/hosting.json` — hosting configuration

## Local development

Install dependencies, then run the development script defined in `package.json`.

Open the `notes` directory in Obsidian using **Open folder as vault**.

## GitHub workflow

Use short-lived branches prefixed with `codex/`, keep product decisions in `notes/Decision Log.md`, and open a pull request for review before merging into the default branch.
